Ersilia Model Hub

The Ersilia Model Hub is a free, open-source repository of Artificial Intelligence and Machine Learning (AI/ML) models for drug discovery.

¬†

This presentation is a guide on how to install and use a model from the Hub.

Pre-requisites

Packages and environments that are needed before installing Ersilia include:

  • Linux or Mac OS¬†
  • Python (Python 3.7 or later)
  • Conda
  • Docker
  • Git and Git LFS

¬†

* For WSL, refer to our detailed installation guidelines

Step 1

The next step is to create a Conda environment for Ersilia, activate it and install Ersilia.

# create a conda environment
conda create -n ersilia python=3.7

# activate the environment
conda activate ersilia

# clone from github
git clone https://github.com/ersilia-os/ersilia.git
cd ersilia

# install with pip (use -e for developer mode)
pip install -e .

Step 3

After some time, Ersilia should be installed.

¬†

To check that Ersilia CLI works, see Ersilia help options.

# see ersilia cli options
ersilia --help

# the output

Step 4

Isaura lake is a package that helps cache your model predictions (store predictions in your local computer). It is highly recommended to download it as it is a relatively light Python package.

# clone from github
git clone https://github.com/ersilia-os/isaura.git
cd isaura

# install with pip (use -e for developer mode)
pip install -e .

How to fetch a model from the Hub

You can explore models in the Ersilia Model Hub. Models that calculate or predict properties of molecules, for example, exist in the Hub amongst others.

An example model is molecular weight which calculates the molecular weight of your molecule of choice.¬†

# download and install
ersilia fetch molecular-weight

Use the API

Each model has API endpoints. The API endpoint of interest for molecular weight is to calculate the molecular weight of a molecule.

¬†

Molecules are to be written in SMILES format.¬†

# calculate molecular weight of Aspirin using SMILES
ersilia api calculate -i "CC(=O)OC1=CC=CC=C1C(=O)O"

# the output
{
    "input": {
        "key": "BSYNRYMUTXBXSQ-UHFFFAOYSA-N",

        "text": "CC(=O)OC1=CC=CC=C1C(=O)O"
    },
    "output": {
        "mw": 180.15899999999996
    }
}

Close the service

When you're done using a model, you have to close it.

# close the service
ersilia close
